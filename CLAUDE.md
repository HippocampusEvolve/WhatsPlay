# WhatsPlay

Мобильное приложение «Чем занять детей»: родитель указывает возраст детей и контекст (дом/улица, город/деревня), приложение выдаёт карточку-игру из оффлайн-базы. Игры собраны из разных стран, городов и эпох («так играли дети в …»).

## Пользователь

- Общается по-русски, не программист — объяснять просто, без жаргона.
- Телефон — Android: приоритет сборки APK; iOS — позже (нужен Mac/облако).

## Стек и окружение

- Flutter (stable, C:\flutter), интерфейс: русский + английский (flutter_localizations, ARB-файлы в `lib/l10n/`).
- Windows 10 Home; ANDROID_HOME=`%LOCALAPPDATA%\Android\Sdk`, JAVA_HOME=Android Studio jbr (JDK 21).
- База игр — оффлайн JSON в `assets/activities/` (никакого LLM внутри приложения, никаких API-ключей).

## Команды

- Тесты: `flutter test`
- Анализ: `flutter analyze`
- Сборка APK для телефона: `flutter build apk --debug` (тест) / `flutter build apk --release` (итог)
- Готовый APK: `build/app/outputs/flutter-apk/`

## Правила

- НИКОГДА не коммитить секреты: `key.properties`, `*.jks`, `*.keystore`, токены (уже в .gitignore).
- Репозиторий: https://github.com/HippocampusEvolve/WhatsPlay — пушить после каждого завершённого рабочего куска.
- После значимых изменений обновлять PROGRESS.md (что сделано / что дальше).
- Каждая игра в базе обязана иметь: возраст от–до, место, число игроков, реквизит, происхождение (страна/эпоха), пометку безопасности если нужна.
- Тексты игр — на русском и английском (поля `_ru` / `_en` в JSON).
